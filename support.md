## 자주 묻는 질문(FAQ)

### 1) 명함 등록은 어디에 저장되나요?
기본적으로 **iOS 연락처(주소록)에 자동 저장하지 않고**, 앱 내부 목록에만 저장됩니다.  
필요할 때만 **선택한 명함을 iOS 연락처로 내보내기**할 수 있습니다.

### 2) “등록”을 눌렀는데 카메라가 안 열리거나 앱이 종료돼요.
iOS는 권한 문구(Info.plist)가 없으면 카메라 기능을 사용할 때 앱이 종료될 수 있습니다.

- **설정 확인**
  - iPhone 설정 → 개인정보 보호 및 보안 → 카메라 → **명함관리 허용**
  - iPhone 설정 → 개인정보 보호 및 보안 → 사진 → **명함관리 허용**

### 3) 스캔은 됐는데 글자 인식(OCR)이 이상해요.
명함 배경/조명/각도에 따라 OCR 정확도가 크게 달라질 수 있습니다.

- **개선 팁**
  - 빛 반사를 피하고 밝은 곳에서 촬영
  - 명함이 화면에 꽉 차도록 정면에서 촬영
  - 흐림이 없도록 초점이 맞은 상태에서 촬영
  - 가능하면 “명함 스캔(문서 스캐너)” 기능 사용

### 4) 이메일이 `@` 대신 `a`로 인식돼요.
OCR에서 흔히 발생하는 오류입니다. 앱은 자동 보정을 시도하지만, 100% 완벽하긴 어렵습니다.

- **권장**
  - 결과 화면에서 이메일을 한 번 확인 후 저장
  - 지속적으로 특정 명함에서 반복되면, “OCR 원문” 일부를 문의로 보내주시면 개선에 도움이 됩니다.

### 5) 명함 앞/뒤(뒷면)도 저장/파싱할 수 있나요?
내 명함(My Card)에서는 **앞면/뒷면 이미지**를 저장할 수 있으며, 뒷면도 OCR/파싱하여 별도로 저장합니다.  
내 명함 화면에서 **이미지를 좌우로 넘기면(앞/뒤)** 해당 면의 파싱 결과가 함께 표시됩니다.

### 6) 공유는 어떻게 하나요?
내 명함은 아래 방식으로 공유할 수 있습니다.
- vCard(.vcf) 공유
- QR 코드 공유(필요 시 명함 이미지 포함 공유)
- 텍스트 공유
- 명함 이미지 공유(현재 보고 있는 면 기준)

## 개인정보 및 권한 안내

### 처리 방식
- 명함 이미지 스캔/텍스트 인식(OCR) 결과는 **기기 내에서 처리**되며, 별도 서버로 전송하지 않는 구조를 목표로 합니다.

### 권한이 필요한 이유
- **카메라**: 명함 촬영/스캔을 위해 필요
- **사진**: 사진에서 명함 이미지를 선택할 때 필요
- **연락처(선택)**: 사용자가 “연락처로 내보내기”를 선택한 경우에만 필요
